# Lexique : Voltamétrie Cyclique

- **WE (Working Electrode)** : Électrode de travail où se produit la réaction d'intérêt.
- **CE (Counter Electrode)** : Électrode auxiliaire permettant de fermer le circuit électrique.
- **RE (Reference Electrode)** : Électrode au potentiel stable servant de référence.
- **Voltammogramme** : Courbe $I = f(E)$ obtenue lors d'un cycle de potentiel.
- **Surtension ($\eta$)** : Écart entre le potentiel appliqué et le potentiel d'équilibre.
- **Double Couche Capacité** : Phénomène de stockage de charge à l'interface métal/solution.
- **Scan Rate ($v$)** : Vitesse de balayage du potentiel [V/s].
- **Réversible** : Se dit d'un système où le transfert d'électrons est très rapide devant le transport de masse.
